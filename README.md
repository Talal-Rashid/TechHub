# 🛠️ TechHub — Multi-Role Tech Marketplace App

**TechHub** is a multi-role e-commerce Android application for tech products. It features role-based access, admin-controlled user approval, Firebase backend integration, and upcoming tools like a PC Builder and Phone Finder.

---

## 🚧 Current Status

Actively under development. Progress tracked below.

### ✅ Completed Features

- ✅ Static page navigation setup
- ✅ Firebase authentication and Firestore integration
- ✅ Role-based login system
- ✅ Admin panel for account approvals
- ✅ Hardcoded fallback admin account

### ⏳ Upcoming Features

- ⏳ Email notification on approval
- ⏳ User profile view and update
- ⏳ Vendor product upload system
- ⏳ Product categories with filters
- ⏳ PC Builder (component compatibility logic)
- ⏳ Phone Finder (GSMArena-style search)
- ⏳ Order system (cart, checkout, tracking)
- ⏳ Real-time chat between users and vendors
- ⏳ Push notification integration

---

## 👤 User Roles

| Role   | Access Description                          |
|--------|---------------------------------------------|
| User   | Register, browse products, place orders     |
| Vendor | Upload and manage product listings          |
| Admin  | Approve users/vendors, manage platform data |

---

## 🧰 Tech Stack

- **Language:** Kotlin
- **UI Framework:** Android XML
- **Architecture:** MVVM (ViewModel + LiveData)
- **Backend:** Firebase (Auth, Firestore, Storage planned)
- **Others:** Navigation Component, Firebase Cloud Messaging (planned)
